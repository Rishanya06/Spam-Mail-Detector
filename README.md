# Spam Mail Detector

## Project Overview

This project uses Machine Learning and Natural Language Processing (NLP) techniques to classify SMS messages as Spam or Ham (Non-Spam).

## Dataset

SMS Spam Collection Dataset

## Technologies Used

* Python
* Pandas
* NLTK
* Scikit-learn
* Matplotlib
* Seaborn

## Workflow

1. Load SMS dataset
2. Preprocess text data
3. Remove stopwords and punctuation
4. Convert text into TF-IDF vectors
5. Split data into training and testing sets
6. Train Multinomial Naive Bayes classifier
7. Evaluate model performance

## Results

* Accuracy: 97.04%
* Precision (Spam): 100%
* F1 Score (Spam): 88%
